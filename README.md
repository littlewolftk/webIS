命名方式：
-----

#### 关键字开头：

##### 前两位：特性（标签或其他新特性）

* 00: form新特性

##### 中间两位：环境

* 00: 用户点击
* 01: XX浏览器XX版本（可改动）

##### 后一位：漏洞类型

* 0: XSS
* 1: CSRF

-----

###### 注：

两位数字用十六进制表示

------
